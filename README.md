# Textbooks In Primo (TIP)

Work in progress. Stay tuned...

## Requirements

1. A .csv export of courses and their respective textbooks from your bookstore each term that has fields for:
	- Textbook Title
	- Textbook ISBN Number
	- Course Name or Code
	- Course Section
	- Course Professor Name
	- Course Professor Email

2. A web server configured to run PHP scripts.

3. A MySQL server to store the data.

## Setup

1. Download this repository's files to your web server.

2. Create MySQL database for TIP.
